# ⚡ 霍格沃茨婚礼挑战赛 (Hogwarts Wedding Challenge)

一个哈利波特主题的婚礼派对游戏 Web App，适合投影到大屏幕上使用。

## 🎮 游戏规则

1. 三队各派一人上场，**不能看大屏幕**
2. 大屏幕随机抽取任务动作，队友用 **YES / NO** 引导（靠近目标 = YES，远离目标 = NO）
3. 完成动作用时最短的队伍获胜，分别获得 **3 / 2 / 1** 积分

## 🏰 功能特性

- 哈利波特主题 UI（深邃夜空 + 金色魔法风格）
- 三队对应霍格沃茨学院：🦁 格兰芬多、🐍 斯莱特林、🦅 拉文克劳
- 随机滚动抽取任务动作（手动停止）
- 计时器（停止后自动记录到选中队伍）
- 两轮积分累计
- 队伍名称可双击编辑
- 全屏规则展示
- 键盘快捷键：空格=计时器，R=抽取，1/2/3=选队

## 🚀 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需任何依赖或服务器。
